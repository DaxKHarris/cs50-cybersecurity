# CS50 Cybersecurity
Working through Harvard's CS50 Introduction to Cybersecurity. This repo documents
my problem set write-ups, scores, and final project as I work through the material.

## Topics Covered
### Lecture 0: Securing Accounts
- Passwords & Two Factor Authentication
- Credential Stuffing
- Social Engineering
- Phishing
- Machine in the Middle Attacks

### Lecture 1: Securing Data
- Hashing
- Symmetric and Asymmetric Cryptography
- Digital Signatures
- Passkeys
- Encryption
- Data Deletion

### Lecture 2: Securing Systems
- HTTP and HTTPS
- Cookies and Supercookies
- IP Addresses
- VPN
- SSH
- Proxy
- Packet Sniffing
- Malware

### Lecture 3: Securing Software
- Code Injection
- Stored Attacks
- SQL Injection
- Client and Server Side Validation
- Cross Site Request Forgery
- Arbitrary Code Execution

### Lecture 4: Preserving Privacy
- VPN
- Tor
- Third Party Cookies
- Tracking Cookies
- Fingerprinting
